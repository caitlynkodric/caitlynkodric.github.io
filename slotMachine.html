The group project was created as a final project in the Systems Design Digital Systems Laboratory (SYDE 192L). In this course we used an Arduino Uno to create circuits and solve problems.
The Slot Machine works with a user input which 'spins' values on an LCD screen. The play enters quarters to the amount they want to bet into the slot to complete the circuit. When the play button is hit, the random number generator spins each column for a certain number of seconds. The output of the program uses a timer for the total time of the game, if no button is pressed the game will reset. When the slot machine shows three identical numbers, the slot machine motor will tilt the collection of coins to the winning player!
